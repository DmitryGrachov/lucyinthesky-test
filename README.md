## Getting Started

Install

```bash
npm install
# or
yarn install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Run tests:

```bash
npm run test
# or
yarn test
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
